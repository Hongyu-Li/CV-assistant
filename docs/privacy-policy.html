Privacy Policy

CV Assistant (简历助手) — Effective Date: March 1, 2026

CV Assistant is a desktop application developed by Cheng Tang for AI-powered resume and CV generation. This privacy policy explains how the application handles your data. We are committed to protecting your privacy — the app is designed so that your data stays on your device.

1. Information Collection

CV Assistant does not collect, transmit, or store any personal data on external servers controlled by the developer. The app does not use cookies, analytics, tracking pixels, or any form of user tracking.

The only information you provide is entered voluntarily within the app:

  • Your personal profile information (name, work experience, education, skills, etc.) used to generate resumes.
  • Job descriptions you paste into the app for tailored resume generation.
  • API keys for third-party AI providers that you choose to configure.
  • App settings and preferences.

2. Data Storage

All data is stored locally on your device. By default, data is saved in the ~/.cv-assistant/ directory (or a workspace directory you configure). This includes:

  • Your profile data (profile.json, profile.md)
  • Generated resumes (stored in the resumes/ subdirectory)
  • Application settings, including any API keys you configure

No data is uploaded to any server owned or operated by the developer. You have full control over your data and can delete it at any time by removing the workspace directory.

3. Third-Party Services

When you generate a resume, CV Assistant sends your profile data and the job description to the third-party AI provider you have selected. This is the only network communication initiated by the app (aside from update checks described below).

The app supports the following third-party AI providers:

  • OpenAI
  • Anthropic
  • Google Gemini
  • DeepSeek
  • Ollama (local)
  • OpenRouter
  • Groq
  • Mistral
  • Qwen (Alibaba)
  • Zhipu (GLM)
  • Kimi (Moonshot)
  • Custom endpoints

Each provider has its own privacy policy and terms of service. You are responsible for reviewing the privacy practices of any provider you choose to use. The developer of CV Assistant has no control over how third-party providers handle data sent to their APIs.

API keys are stored locally on your device and are never transmitted to the developer. They are sent only to the respective AI provider's API endpoint when you initiate a resume generation request.

Additionally, CV Assistant checks for application updates via GitHub Releases. This check contacts GitHub's servers but does not transmit any personal data.

4. Data Security

Because all data is stored locally on your device, its security depends on your device's own security measures (disk encryption, login password, etc.). We recommend:

  • Keeping your operating system and the application up to date.
  • Using your operating system's built-in disk encryption.
  • Being cautious about which third-party AI providers you share your data with.

5. Children's Privacy

CV Assistant is not directed at children under the age of 13. We do not knowingly collect personal information from children. Since all data remains on your local device and is never transmitted to us, there is no mechanism through which we could collect such data.

6. Changes to This Policy

We may update this privacy policy from time to time. Changes will be posted on this page with an updated effective date. We encourage you to review this policy periodically. Continued use of the application after changes constitutes acceptance of the updated policy.

7. Contact

If you have any questions or concerns about this privacy policy, please open an issue on our GitHub repository:

https://github.com/tc9011/CV-assistant

隐私政策

简历助手 (CV Assistant) — 生效日期：2026 年 3 月 1 日

简历助手是由 Cheng Tang 开发的桌面应用程序，用于借助 AI 生成简历。本隐私政策说明了本应用如何处理您的数据。我们致力于保护您的隐私——本应用的设计确保您的数据始终留在您的设备上。

一、信息收集

简历助手不会收集、传输或将任何个人数据存储到开发者控制的外部服务器上。本应用不使用 Cookie、分析工具、追踪像素或任何形式的用户追踪。

您提供的信息均为在应用内自愿输入：

  • 您的个人档案信息（姓名、工作经历、教育背景、技能等），用于生成简历。
  • 您粘贴到应用中的职位描述，用于生成定制简历。
  • 您选择配置的第三方 AI 服务商的 API 密钥。
  • 应用设置和偏好。

二、数据存储

所有数据均存储在您的本地设备上。默认情况下，数据保存在 ~/.cv-assistant/ 目录中（或您配置的工作区目录）。包括：

  • 您的个人档案数据（profile.json、profile.md）
  • 生成的简历（存储在 resumes/ 子目录中）
  • 应用设置，包括您配置的任何 API 密钥

不会有任何数据被上传到开发者拥有或运营的服务器。您对自己的数据拥有完全控制权，随时可以通过删除工作区目录来删除所有数据。

三、第三方服务

当您生成简历时，简历助手会将您的个人档案数据和职位描述发送到您所选择的第三方 AI 服务商。这是本应用发起的唯一网络通信（除下文描述的更新检查外）。

本应用支持以下第三方 AI 服务商：

  • OpenAI
  • Anthropic
  • Google Gemini
  • DeepSeek
  • Ollama（本地）
  • OpenRouter
  • Groq
  • Mistral
  • 通义千问（阿里巴巴）
  • 智谱（GLM）
  • Kimi（月之暗面）
  • 自定义端点

各服务商有各自的隐私政策和服务条款。您有责任审查所选服务商的隐私惯例。简历助手的开发者无法控制第三方服务商如何处理发送至其 API 的数据。

API 密钥存储在您的本地设备上，绝不会传输给开发者。密钥仅在您发起简历生成请求时发送到相应 AI 服务商的 API 端点。

此外，简历助手会通过 GitHub Releases 检查应用更新。此检查会连接 GitHub 的服务器，但不会传输任何个人数据。

四、数据安全

由于所有数据均存储在您的本地设备上，其安全性取决于您设备自身的安全措施（磁盘加密、登录密码等）。我们建议：

  • 保持操作系统和本应用的更新。
  • 使用操作系统内置的磁盘加密功能。
  • 谨慎选择与您共享数据的第三方 AI 服务商。

五、儿童隐私

简历助手不面向 13 岁以下的儿童。我们不会故意收集儿童的个人信息。由于所有数据均保留在您的本地设备上且从未传输给我们，因此不存在我们收集此类数据的途径。

六、政策变更

我们可能会不时更新本隐私政策。变更将发布在本页面并更新生效日期。我们建议您定期查阅本政策。在变更后继续使用本应用即表示您接受更新后的政策。

七、联系方式

如果您对本隐私政策有任何疑问或顾虑，请在我们的 GitHub 仓库中提交 Issue：

https://github.com/tc9011/CV-assistant
